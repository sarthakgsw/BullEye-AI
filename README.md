# BullEye AI 🚀

BullEye AI is an AI-powered Indian Stock Market Assistant designed to help traders and investors with smart market analysis.

## Planned Features

- 📈 Intraday Buy/Sell Signals
- 📊 Swing Trading Analysis
- 💹 Long-Term Investment Analysis
- 📉 Technical Indicators (RSI, MACD, EMA, VWAP)
- 🕯️ Candlestick Pattern Recognition
- 📰 News & Sentiment Analysis
- 🤖 AI-Based Decision Making
- 🛡️ Risk Management
- 📋 Portfolio Tracking
- 🔔 Smart Alerts

## Tech Stack

- Stock Market APIs
